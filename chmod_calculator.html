Permissions

  • -rw-r--r-- (644)
  • dr-------- (400)

    First character

    • - File.
    • d repository.
    • l symbolic link.

    Other characters

    • r for Read (4).
    • w for Write (2).
    • x for Execute (1).
    • 7 for all rights.
    • First 3 characters for the owner's permissions.
    • Second 3 characters for the group's permissions.
    • Third 3 characters for the other's permissions.
